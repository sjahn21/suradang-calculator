# 수라당 파렛트 분배 계산기 - 최적화된 프롬프트

## 📋 한 번에 요청하는 완벽한 프롬프트

```
당신은 세계 최고 수준의 프로그래머이고 웹 프로그램 설계자입니다.

수라당에서 사용하는 파렛트 분배 계산용 웹앱을 만들어 주세요.

## 핵심 기능

### 입력
1. 품목당 총 수량 (예: 12000개)
2. 1 파렛트당 떡 개수 (기본값: 100개)
3. 줄수와 파렛트 수 분배 방식 선택:
   - **스마트 분배**: 한 줄당 20~23판 범위에서 자동 최적 분배
   - **줄 수 우선**: 원하는 줄 수 입력 → 한 줄당 파렛트 수 자동 계산
   - **파렛트 수 우선**: 한 줄당 파렛트 수 입력 → 줄 수 자동 계산

### 출력
1. 선택한 분배 모드 표시
2. 한 줄당 파렛트 수
3. 채운 줄 + 마지막 줄 (예: "총 6줄 + 17판" - "총"은 작은 폰트)
4. 마지막 [N]판 떡 최소 수량 (N은 마지막 줄의 판 수)

### 계산 로직
- 스마트 분배: 20~23판 중 마지막 줄이 가장 많이 채워지는 값 선택
- 줄 수 우선: 총 파렛트 수 ÷ 원하는 줄 수 = 한 줄당 파렛트 수
- 파렛트 수 우선: 사용자 지정값 사용

## UI/UX 요구사항

### 디자인
- 2025년 최신 트렌드의 모던한 디자인
- 그라디언트 배경 (보라색 계열)
- 글래스모피즘 효과의 흰색 카드
- 부드러운 애니메이션

### 분배 방식 선택 UI
- 3개 옵션을 가로로 나란히 배치 (모바일은 세로)
- 라디오 버튼 형식
- 선택 시 보라색 그라디언트 강조
- "줄 수 우선"과 "파렛트 수 우선" 선택 시:
  - 해당 옵션 카드 아래에 입력 필드가 자연스럽게 확장
  - 카드와 입력 필드가 하나로 연결된 느낌
  - 입력 필드는 서브메뉴처럼 작고 컴팩트하게 (최대 너비 140px)
  - 통합된 테두리와 배경색으로 시각적 연결

### 반응형
- 아이폰, 안드로이드 완벽 대응
- PWA 스타일 (설치 불필요)
- 터치 친화적

### 편의 기능
- Enter 키로 계산
- 리셋 버튼 (모든 입력 초기화, 스마트 분배로 복귀)
- 입력 오류 검증 및 친절한 에러 메시지

## 기술 요구사항
- 순수 HTML, CSS, JavaScript (외부 라이브러리 없음)
- 단일 파일 (index.html)
- 즉시 실행 가능
- GitHub Pages, Netlify 등에 바로 배포 가능

## 브랜딩
- 타이틀: "🥮 수라당 파렛트 분배 계산기"
- 서브타이틀: "파렛트 줄수와 판수 자동 계산"
- 푸터: "© 2025 수라당 | 파렛트 분배 계산기"

## 예시 시나리오
- 12000개 생산, 100개/파렛트, 스마트 분배 → 자동으로 20~23판 중 최적값 계산
- 11000개 생산, 100개/파렛트, 줄 수 우선(6줄) → 한 줄당 19판으로 계산
```

## 🎯 프롬프트 작성 팁

### 잘한 점
1. **명확한 역할 부여**: "세계 최고 수준의 프로그래머"
2. **구체적인 예시**: 숫자와 시나리오 제공
3. **단계적 요구사항**: 입력 → 계산 → 출력 순서
4. **UI/UX 세부사항**: 색상, 크기, 애니메이션까지 명시
5. **기술 제약**: 단일 파일, 순수 JS 등

### 추가하면 좋았을 것
1. **우선순위 표시**: "필수", "선택" 구분
2. **참고 이미지**: 원하는 디자인 스타일 이미지
3. **에러 케이스**: 예외 상황 처리 방법
4. **접근성**: 키보드 네비게이션, 스크린 리더
